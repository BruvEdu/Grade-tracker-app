# 🍎 Pre-K & Above Grade Tracker
A simple Python tool to help educators filter passing scores and calculate class averages.

## How it works
- Filters out any grades below **60**.
- Calculates the **average** of the remaining passing grades.
- Prevents errors if no students passed.

## How to Run
1. Install [Python](https://www.python.org/).
2. Run `python grade_tracker.py` in your terminal.
